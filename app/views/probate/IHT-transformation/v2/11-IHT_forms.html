Which forms did you submit to HMRC?

Using the Inheritance Tax checker is not the same as submitting a form to HMRC. The checker will give you a result straight away. If you have only used the Inheritance Tax checker, select that you did not have to submit a form.

If you submitted the IHT400 or the IHT400 and IHT421 you will have had to wait for a reply from HMRC.
